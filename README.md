### 💪 2. Body Performance Analytics – ML Classification & Regression

> **Predict physical fitness level (A–D)** and **broad jump distance** using machine learning.

- **Data:** 13,393 individuals, 12 physiological features.
- **Cleaning:** Removed outliers (IQR), handled impossible values (BP=0, flexibility 213cm).
- **EDA:** Correlation analysis – flexibility (+0.59) and sit‑ups (+0.45) are top predictors.
- **Models tested:** SVM (RBF), Decision Tree (tuned depth), KNN (tuned k), Neural Network (2 hidden layers), Linear Regression.
- **Best classification accuracy:** ~72% (SVM & Neural Network).  
  **Best regression R²:** ~0.79 (Linear Regression).

📎 [Repository Link]([https://github.com/mostafaeltawcel/body-performance-analytics])
